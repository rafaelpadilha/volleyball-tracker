### Volleyball Tracker
O objetivo deste projeto é analizar imagens (vídeos/webcam streaming) e calcular a velocidade da bola. 


## Requisitos

* Python 3.13

(Opcional) Criar venv
```bash
python -m venv <nome>
```

```bash
python -m pip install -r requirements.txt
```

## Executando

```bash
python main.py
```
